2. 변수 생성 방법
    (1) 변수란 ? 데이터를 담는 공간
    (2) 변수 선언 : 데이터를 담을 공간을 생성하는 것
        변수 초기화 : 생성된 변수에 데이터를 전달하는 것
    (2-1) js는 문장이 끝났을 때 ';'을 반드시 입력해준다.
    (3) console.log() : 변수 안에 데이터를 확인할 때 사용하는 명령어
    (4) 변수 생성 시 주의사항 :
        1) 변수 이름 작성시 숫자가 맨 앞에 사용될 수 없음
            var 1str; (X)
        2) 변수 이름은 최대한 자세하게 작성
            var randomNumber;
        3) 의미가 불명확한 단어들의 조합은 피해야함
            var tmax;
