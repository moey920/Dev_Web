레이아웃에 영향을 미치는 속성
------------------------------------------------------------------------------------------------------------------------------------
1. display
    (1) Block과 Inline 요소의 성격을 바꿀 때 사용한다. inline-block을 사용하면 두 요소의 성격을 모두 가진다.
    (2) p {display:inline;}
        a {display:block;}
        a {display:inline-block;}
------------------------------------------------------------------------------------------------------------------------------------
2. float
    (1) 선택자를 띄워 새로운 레이어층을 형성한다.
    (2) 왼쪽 정렬, 오른쪽 정렬을 할 때 사용한다(정렬)
    (3) head 태그의 style 태그 내에, abc {float:left;}와 같이 사용한다.
    (4) '공중에 띄우다'라는 기능을 가지고 있다. float 사용 후 clear하지 않으면 레이어층이 겹치는 현상이 발생한다.
        def {clear : both;}를 사용한다.
        - float과 clear는 함께 다닌다.
        - float을 마지막으로 사용하는 지점, 다음 태그에 clear를 입력한다.
        - clear는 left, rigth, both 세 가지의 속성을 갖는다. (both값은 양쪽에 대한 모든 기능을 꺼준다. both만 기억할 것.)
------------------------------------------------------------------------------------------------------------------------------------
3. 브라우저와 공간 사이의 공백 제거
    (1) html, body 태그는 margin, padding 값을 가지고 있으므로 초기화 한 후 사용해야한다.

        <style>
            html, body {
                margin : 0px;
                padding : 0px;
            }
        </style>

        혹은(이 방법을 추천한다. 다양한 태그가 margin,padding 초기 값을 가지고 있기 때문에!)
        <style>
            * { /* 모든 태크의 margin, padding 값을 0으로 초기화 */
                margin : 0px;
                padding : 0px;
            }
        </style>
